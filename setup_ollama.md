# Configuración de Ollama (Modelo Gratuito)

## Paso 1: Instalar Ollama
1. Descarga Ollama desde: https://ollama.com/download
2. Instala y ejecuta la aplicación Ollama

## Paso 2: Descargar un modelo
Ejecuta en PowerShell (una vez instalado Ollama):
```powershell
ollama pull llama3.2
```

O puedes probar otros modelos gratuitos:
- `ollama pull mistral` (más rápido)
- `ollama pull qwen2.5` (muy bueno)
- `ollama pull phi3` (pequeño y eficiente)

## Paso 3: Ejecutar tu código
```powershell
python main.py
```

¡Listo! Tu agente funcionará 100% gratis con modelos locales.
